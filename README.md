## Getting Started

1. Install dependencies:
   - `npm install`
2. Start the development server:
   - `npm run dev`

## Available Scripts

- `npm run dev` - run the app locally
- `npm run build` - create a production build
- `npm run preview` - preview the production build
- `npm run lint` - run lint checks
